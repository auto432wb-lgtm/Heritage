# The Heritage Center — Environment Prompts v1

## Purpose
Generation-ready prompts for the recurring pilot sets. These should be used to build the visual foundation of the series before heavy scene-specific generation begins.

## Global Style Lock
Use this line across all environment generations:

> mockumentary TV comedy set design, semi-realistic stylized live-action look, grounded community-center realism, slightly cheap institutional aesthetic, clear readable composition, not overly cinematic, not too glossy, not futuristic, not cluttered beyond readability, built for recurring dialogue scenes and mockumentary coverage

---

## Prompt
> interview setup for a low-budget pseudo-academic institution called The Heritage Center, mockumentary TV comedy set design, seated confessional background, Heritage Center banner or institutional backdrop, slightly awkward but professional-looking setup, simple chair, controlled lighting that tries to look official but feels low-budget, readable composition, neutral wall or banner background, built for repeated character interviews, grounded realism, semi-realistic stylized live-action look, clear and reusable, not too busy

---

## Practical Rule
For each environment, generate:
- 1 strong master image first
- then 1 alternate angle only if needed
- then a scene variant later

Do not over-generate environments before the first cut exists.
