# The Heritage Center — Environment / Room List v1
## Episode 1 Production Sets

## Purpose
This file identifies the recurring visual spaces needed to build the pilot efficiently. The goal is to create reusable sets that support multiple scenes, interviews, and pickups without overbuilding the world.

---

## 1. Main Classroom
### Used in
- Cold Open
- Scene 2
- Scene 6
- parts of Scene 7

### Function
The main institutional performance space. This is where group scenes, fake teaching, rehearsals, and public-facing pseudo-education happen.

### Visual requirements
- folding tables or low-budget classroom furniture
- dry-erase board / whiteboard
- Heritage Center signage
- accreditation sign / fundraiser sign variants
- maps, posters, prop papers, markers, ring light spill, clutter
- enough depth for ensemble blocking
- room for a camera to feel mockumentary and observational

### Design rule
Should feel like a community-center classroom trying to cosplay as an academic institution.

### Need to generate
- master wide
- front-of-room angle
- board-side angle
- back-of-room angle
- donor-preview/rehearsal variant

## 2. A1’s Office / Accreditation Room
### Used in
- Scene 1
- Scene 8

### Function
A cramped command center where pseudo-ideas are forced into institutional paperwork.

### Visual requirements
- desk with binders and labeled folders
- laptop / papers / accreditation packet props
- printer or filing trays if possible
- cramped but organized feel
- visual contrast with the chaos of the classroom

### Design rule
Should feel like one man is trying to bureaucratize nonsense in a room too small for his ambition.

### Need to generate
- master medium-wide
- desk-side angle
- over-shoulder document angle
- low-energy nighttime aftermath variant

## 3. Livestream Room
### Used in
- Scene 5
- Scene 7

### Function
The Center’s most dangerous room: where pseudo-scholarship becomes direct-to-audience performance.

### Visual requirements
- ring light
- desk setup
- cheap fake books / academic-looking props
- taped or crooked Heritage Center logo backdrop
- streaming mic / phone stand / camera setup
- enough space for multiple people to crowd into frame

### Design rule
Should feel like a homemade media room pretending to be a broadcast institution.

### Need to generate
- master wide
- on-camera host angle
- off-camera side angle
- crowded live-event variant

## 5. Hallway / Common Area
### Used in
- Scene 4
- spillover transitions
- future passing scenes / tension beats

### Function
A connective space where side conversations, low-budget institution signage, and transitional comedy happen.

### Visual requirements
- walls with taped notices
- low-rent institutional hallway feel
- bulletin board or posted signage
- enough empty space for two-person walk-and-talks or interruptions

### Design rule
Should feel like a real building that is trying very hard to look official on no money.

### Need to generate
- hallway master
- notice-board angle
- doorway / passing angle

---

## 6. Interview Setup
### Used in
- all confessionals

### Function
The anchor of the mockumentary style. This space gives the show consistency and rhythm.

### Visual requirements
- seated chair setup
- Heritage Center banner or background element
- slightly awkward institutional framing
- controlled lighting that tries to look professional but isn’t perfect

### Design rule
Should feel like they set up an official interview corner themselves and are proud of it.

### Need to generate
- neutral interview master
- cleaner A1 version if needed
- same framing adaptable across characters

## Production Rule
These rooms should be simple, repeatable, and slightly cheap-looking. The show’s value comes from performances, props, signs, interviews, and contradictions — not elaborate environment spectacle.
